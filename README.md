# 🌞 Heliocentric Solar System Model 🌌

An **interactive, animated solar system** built with **HTML, CSS, and JavaScript**!  
Explore the planets orbiting around the Sun and hover over them to see their names. 🚀🪐

---

## ✨ Features

- 🌞 **Pulsating Sun** with a glowing effect  
- 🪐 **Planets with realistic orbits**  
- 🌍 **NASA-inspired planet images** (stored locally for fast loading)  
- 🖱️ **Hover effects** with planet labels  
- 📱 **Responsive design** for mobile and desktop  
- ⭐ Optional **starfield background** for extra space vibes
